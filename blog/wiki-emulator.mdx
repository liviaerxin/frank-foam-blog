# Wiki Emulator

[Configure hardware acceleration for the Android Emulator](https://developer.android.com/studio/run/emulator-acceleration)

Hardware acceleration:

- Graphics acceleration: Improve screen rendering, especially in games
- VM acceleration(a.k.a, CPU acceleration): Improve execution speed

Graphics acceleration can be enabled with `-gpu` option.
Windows: It may need you add the `emulator.exe` to the `Graphics settings`, such as `"C:\Users\xx\AppData\Local\Android\Sdk\emulator\emulator.exe"`

VM acceleration is enabled in default.
For instance, if `WHPX` is configured in windows, it will be used automatically. 

```sh
emulator.exe -avd Pixel_3a_API_34_extension_level_7_x86_64 -gpu host -no-snapshot
```
